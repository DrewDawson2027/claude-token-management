# Code of Conduct

## Our Standard

This project expects direct, technical, respectful collaboration.

Examples of behavior that help:

- bringing reproducible evidence
- challenging claims without attacking people
- keeping discussion tied to the problem being solved
- sharing concrete failure cases, benchmarks, or patches

Examples of unacceptable behavior:

- harassment, insults, or intimidation
- bad-faith dogpiling
- posting private information, secrets, or sensitive logs
- derailing technical threads with personal attacks

## Enforcement

Maintainers may remove comments, close threads, or block participation when conduct damages the project or the people working in it.
